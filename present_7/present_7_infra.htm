IT Use in Indoresco

 

  National Sales Manager:

  • Didn’t have internet, correspond with buyer through telephone and fax machine. If  he wanted to use internet, he must go to International Marketing Manager’s room.

  • Using Microsoft Excell to made simple sales calculation.

  • International Marketing Manager.

  • Has Internet. Corresponding with international buyers through e-mail, telephone and fax machine

  • Computer used only to make letter with word processor (MsWord). To make job done faster.

  Sales Administration:

  • Make invoice using custom made software that was built using Foxpro. The program interface work only by input product code and quantity sold.

  Finance:

  • Generate and entry cash journal using old lotus software.

  Warehouse:

  • Using custom made software to document inventory stock. Inventory stack was update by warehouse employee.

  Production:

  • There is no computer involve in doing the job.
