PROPOSED IT DEVELOPMENT

 
        IT in proposed product is not using full IT development in vision process. However IT development was implemented to be directed in IT development. However online entry order and online payment is not installed yet. Purchase order is still process manual, and payment is still conducted conventional.  
        OLAP and knowledge management is not implemented yet. But improving database management and using spreadsheet software to conduct forecasting and product scheduling. Online database is installed so there data integration between department is maintain.
        Based on suggestion from management, we also improve IT in the area:

  • Implement internet and improve intranet (LAN)

  • Install reliable software (genuine) not piracy product

  • Apply IT for decision making and communication

  • Replace or train employees that could match with IT development
        Information Technology vision process is a blueprint for further development of IT in Indoresco. Therefore IT development in proposed process is toward this blueprint.
